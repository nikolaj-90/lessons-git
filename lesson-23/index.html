Оператори

Варіант №1 console.log('35' + - "22"); // Поверне 13 ?
Варіант 1 – не вірно, оператор "+" тільки з'еднає два рядки із " - " і поверне рядок 35-22, тому що рядок "35" залишається рядком а не числом. Щоб вирахувати 13 потрібно до "35" поставити "+" - це конвертує його в число так же як і - "22". (" - " дасть від'ємне значення).
Вариант №2 console.log('35' * "22"); // Поверне 770 ?
Варіант 2 – вірно, оператор множення переведе типи данних із string в number і порахує їх. Із такими двума видами лапок буде працювати, головне що закриті тикими же.
Вариант №3 console.log('558' > 22++); //Поверне true ?
Варіант 3 – викличе помилку, тому що інкремент заборонено використовувати із числами. Тільки зі змінними.
Вариант №4 let usersCounter = 0; let newUsers = usersCounter++; console.log(newUsers); // Поверне 1 ?
Варіант 4 – не вірно, тут використовується суфіксний інкремент, userCounter++ поверне newUsers 0 а потім виконає зростання на 1.
Вариант №5 console.log(!false && 11 || 18 && !''); //Поверне 18 ?
Варіант 5 – не вірно. Спочатку виконаються && де повернуть true а потім виконається || порівняння з ліва на право, де в порівнянні два (true) поверне останнє 11(true) що потім відобразиться в консолі.
Вариант №6 let name = 0; console.log(name ?? "Без имени")
Варіант 6 – поверне "0" так як нуль це тип данних number і не є null чи undefined.

Умовні оператори

Вариант №1 if (1 === "1") {console.log('Истина!');} else {console.log('Ложь!');}
Варіант 1 – if(1 === "1") - сурова рівність не конвертує в числа а буде зрівати типи данних, тут типи різні що дає false. Тому цей варіант не поверне console.log("Истина!");
Вариант №2 if (5 == "5") { console.log('Истина!'); } else { console.log('Ложь!'); }
Варіант 2 – вірний, "==" конвертує тип із "string" в "number" і потім зрівняє значення 5 і 5 й виконаеться console.log('Истина!');
Вариант №3 let message = (92 > '11' && 58 < 100) ? 'Истина!' : 'Ложь!' ;console.log(message);
Варіант 3 – поверне "Істина", так як всі умови true.
Вариант №4 if (0) { console.log('Ложь!'); } else if (" ") { console.log('Истина!'); }
Варіант 4 – вірний, в умові if(0) - це false а else if (' ') - є пробіл це true і виконає console.log('Истина!');
